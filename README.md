# Bicycle shop
Site about selling bicycles, I use json file for data, implementation of jquey, ajax...

# Functionality:
  - Dynamic content from json
  - Sorting and filtering with js
  - client validation
  - local storage (shoping cart)

Used technologies:
  - Html
  - Css
  - Js (Jquery)
  - Ajax
  - Bootstrap
